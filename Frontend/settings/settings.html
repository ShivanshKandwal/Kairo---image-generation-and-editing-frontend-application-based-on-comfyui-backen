Host Configuration

This page allows you to configure the core settings for your Kairo host application.

Network Identity

Set a stable, human-readable network name for this host. This allows other devices on your network to connect using a memorable .local address, even if the host's IP address changes.

Your selection is saved and applied after the application restarts.

Active Model

Choose which AI model Kairo uses for its creative tasks. The dropdown menu lists all compatible models found on the host machine.

To add a new model, place its file in the appropriate ComfyUI models folder (e.g., ComfyUI/models/unet/) and restart the Kairo application.
